BabyGrid re-adapted from the original source code (by David Hillard). Contains a dialog based demo of the grid control.<br>
Macro wrappers for SendMessage are added. A refresh macro is also added, that demonstrates how to use it in the WM_SIZE handler to avoid screen flicker.

**Author**: David MacDermot

Updated August 26, 2006 (problems running on Windows 9X)
