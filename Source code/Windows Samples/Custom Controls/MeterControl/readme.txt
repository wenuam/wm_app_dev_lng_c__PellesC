IMPORTANT !
-------------------------------------

This is a full distribution of MeterCtrl32 version 1.5,
intended to be used along with any C99 compiler.

DISTRIBUTION CONTENTS
-------------------------------------

(.) Distribution root directory
   Readme.txt       This file.
   
(.\src) Sources
   meterControl.c  Main source contains function & class overrides.
   meterControl.h  Main source header contains structures & definitions & prototypes.
   
(.\test) Test Application
	mTest.exe		Executable 
	main.c			Test App main source
	main.h			Test App main header
	main.rc			Test App resource file
	mTest.ppj		Pellec project file
	meterControl.c  Main source contains function & class overrides.
    meterControl.h  Main source header contains structures & definitions & prototypes.


(.\dll) dll control dynamic lib
	Meter.dll       dynamic library module
	main.c			DLL main source
	meterControl.c  Main source contains function & class overrides.
    meterControl.h  Main source header contains structures & definitions & prototypes.

(.\dll\test) dll ASM test app MASM32 Compatible
