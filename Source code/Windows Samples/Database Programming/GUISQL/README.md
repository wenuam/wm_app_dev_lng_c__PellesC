ODBC command interpreter - a GUI version of [SQLCmd].

[SQLCmd]: http://www.smorgasbordet.com/pellesc/sample_sqlcmd.htm

Select a database from the ODBC dialog, type a SQL query (for example) in the edit box, and press Alt+Enter. The result is displayed in the ListView.
