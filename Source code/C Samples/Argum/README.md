These programs demonstrates use of argc and argv parameters to print and test through the DOS command line argument list. Tested with Pelles C, v4.00.

- **Argum.c**<br>
This is the World's most powerful example C program - only 3 statements, functions and without other variables except standard parameters - prints DOS command line arguments.<br>
Usage example: argum -R -N -Call

- **Argtest.c**<br>
This example C program tests and prints DOS command line arguments and execute functions.<br>
Usage example: argtest -R -N -Call -O -W -V -S

Just load and compile with Pelles C \*.ppj - files and run on command line with arguments.
