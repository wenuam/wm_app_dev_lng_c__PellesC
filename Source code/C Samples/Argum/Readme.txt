Argum.c
This is the World's most powerful example C program - only 3 
statements, functions and without other variables except standard 
parameters - prints DOS command line arguments.
Usage example:  argum -R -N -Call

Argtest.c
This example C program tests and prints DOS command line 
arguments and execute functions.
Usage example:  argtest -R -N -Call  -O -W -V -S


Just load and compile with Pelles C *.ppj - files and run on command 
line with arguments.
