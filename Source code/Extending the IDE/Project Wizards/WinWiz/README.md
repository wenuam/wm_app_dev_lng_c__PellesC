Sample that demonstrates a Windows GUI application wizard for Pelles C IDE. This is the same source code that is used by the IDE in 'Win32 Application wizard'.

**Setup instructions**:<br>
Build the project. Copy **winwiz.dll** to Pelles-C-install-path**\Bin\Wizards**. Start the IDE and choose **New** -> **Project**. In the list you should see the new wizard.
